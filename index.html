Desenvolvimento de Sistemas

💻 Desenvolvimento de Sistemas

Sobre o Curso

O curso técnico em Desenvolvimento de Sistemas forma profissionais capacitados para planejar, desenvolver e manter sistemas computacionais. Você aprenderá linguagens de programação, banco de dados, lógica e estrutura de software.

O que você vai aprender

  • Programação Web (HTML, CSS, JavaScript)
  • Banco de Dados e SQL
  • Desenvolvimento de Aplicativos
  • Versionamento e Controle de Código

Área de Atuação

Empresas de tecnologia, startups, consultorias de TI e setores de desenvolvimento de software.

📚 Formação de Docentes

Sobre o Curso

O curso técnico em Formação de Docentes prepara profissionais para atuar na educação infantil e anos iniciais do ensino fundamental. É voltado para quem deseja ensinar e inspirar novas gerações.
